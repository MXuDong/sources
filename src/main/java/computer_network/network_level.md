# 应用层
应用层是网络应用程序及他们的应用协议存留的地方。

常见应用层应用：
1. HTTP协议
2. SMTP协议
3. FTP协议
4. DNS系统

### 用途
在一个端系统中的应用程序使用协议与另一个端系统中的应用程序交换分组信息。

我们把这种位于应用层的信息分组成为``报文（message）``
